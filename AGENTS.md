# Для AI-агентов

Если ты работаешь с этим репозиторием ЛОКАЛЬНО (Claude Code и др.):

**Прочитай `_internal/CLAUDE.md` и `_internal/DECISIONS.md` перед любыми
изменениями.** Эти файлы содержат полный контекст проекта, зафиксированные
архитектурные решения и дорожную карту. Они намеренно исключены из публикации
(`.gitignore`) — это рабочие инструкции, не часть сдаваемого проекта.

Если папки `_internal/` нет (ты видишь только публичную версию) — значит контекст
недоступен; не реконструируй его и не выдумывай, запроси у владельца репозитория.
